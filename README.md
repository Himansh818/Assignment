# Admin Dashboard Table


 ## [Live Demo](https://assignmentdashboard.vercel.app/)


]
A React-based admin dashboard table with loading skeleton, sorting, pagination, row selection, and bulk delete functionality.

## Features
- Loading skeleton UI
- Sort by Name & Date
- Pagination (10 rows per page)
- Row selection with Select All
- Bulk delete with localStorage persistence
- Responsive (Table for desktop, Card view for mobile)

## Tech Stack
- React
- Tailwind CSS

## Setup
```bash
npm install
npm run dev

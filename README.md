
  # Admin Dashboard Table


 ## [Live Demo](https://assignmentdashboard.vercel.app/)



A React-based admin dashboard table with loading skeleton, sorting, pagination, row selection, and bulk delete functionality.

## Features
- Loading skeleton UI
- Sort by Name & Date
- Pagination (10 rows per page)
- Row selection with Select All
- Bulk delete with localStorage persistence
- Responsive (Table for desktop, Card view for mobile)


A clean, modern, and responsive Admin Dashboard Table built with React + Tailwind CSS, featuring all essential data-management functionalities such as sorting, pagination, row selection, and local persistence.

This project is designed with a focus on UX, performance, and scalability — suitable for real-world admin panels.

## Tech Stack
- Technology	Purpose
- React.js	Component-based UI
- Tailwind CSS	Fast & modern styling
- LocalStorage	Client-side data persistence

## Installation & Setup

- Clone the repository:

git clone your-repo-link
cd your-folder-name

- Install dependencies:

npm install

- Run development server:

npm run dev

The project will start on:

http://localhost:5173
